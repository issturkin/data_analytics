# Аналитические выводы

В рамках анализа была исследована популярность брендов среди клиентов из Москвы на основе количества заказанных товарных позиций.

В результате анализа для каждого бренда были рассчитаны:

- общее количество заказанных товаров
- доля бренда в общем объёме заказанных товаров среди клиентов Москвы
- ранг бренда по популярности

## Что позволяет увидеть такой анализ

1. **Лидирующие бренды по спросу**  
   Рейтинг брендов показывает, какие бренды наиболее востребованы среди клиентов из Москвы и занимают верхние позиции по количеству заказанных товаров.

2. **Структуру спроса по брендам**  
   Показатель `brand_share_percent` позволяет оценить, насколько равномерно распределён спрос между брендами и есть ли выраженная концентрация на нескольких лидерах.

3. **Приоритетные направления для маркетинга**  
   Полученные результаты можно использовать для выбора брендов, которые имеет смысл:
   - усиливать в рекламных кампаниях
   - продвигать в региональных акциях
   - включать в более детальный анализ спроса по категориям товаров

## Практическая ценность

Такой анализ помогает маркетинговой команде не только определить самые популярные бренды, но и понять их вклад в общий объём заказов. Это делает выводы более полезными для принятия решений, чем простой подсчёт количества заказанных товаров.
