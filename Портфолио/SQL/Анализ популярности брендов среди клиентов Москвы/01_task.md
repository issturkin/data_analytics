# Анализ популярности брендов среди клиентов Москвы

## Бизнес-контекст

Отдел маркетинга хочет понять, какие бренды товаров наиболее популярны среди клиентов из Москвы.  
Первоначальная задача заключалась в том, чтобы определить, какие бренды чаще всего заказывают клиенты из этого города.

Однако для более глубокого анализа недостаточно просто узнать количество заказанных товаров.  
Для принятия маркетинговых решений важно также понять:

- какую долю занимает каждый бренд в общем объёме заказов
- какие бренды являются лидерами по популярности
- насколько сильно спрос сконцентрирован на отдельных брендах

Такой анализ позволяет маркетинговой команде определить ключевые бренды, на которые стоит ориентироваться при планировании рекламных кампаний и работе с ассортиментом.

---

## Задача

Для каждого бренда необходимо рассчитать:

- общее количество заказанных товаров
- долю бренда в общем количестве заказанных товаров среди клиентов Москвы
- рейтинг бренда по количеству заказанных товаров

---

## Используемые таблицы

В анализе используются следующие таблицы:

- `customers` — информация о клиентах  
- `orders` — информация о заказах  
- `order_items` — товары внутри заказов  
- `products` — информация о товарах и брендах  

---

## Требования к результату

Результирующая таблица должна содержать следующие поля:

- `product_brand` — название бренда  
- `items_count` — количество заказанных товаров бренда  
- `brand_share_percent` — доля бренда среди всех заказанных товаров (%)  
- `brand_rank` — ранг бренда по количеству заказанных товаров  

---

## Дополнительные условия

- учитываются только клиенты из города **Москва**
- бренды со значением `NULL` исключаются из анализа
- ранжирование брендов выполняется с помощью оконной функции `DENSE_RANK()`
