# 🎧 Анализ данных об оптовых продажах аудиотехники

## 📋 Бизнес-контекст

Оптовая компания «Карпов Саунд» занимается продажей профессиональной и домашней аудиотехники ведущих мировых брендов. Из-за плановых работ на серверах единое хранилище данных оказалось недоступно, и логи сохранились в виде разрозненных файлов по десяткам разных директорий. Цель данного проекта: собрать единую базу данных из "сырых" файлов, проанализировать динамику заказов, оценить влияние курса валют на покупательскую способность, выявить неликвидные товары на складе и составить рейтинг эффективности менеджеров по продажам.

## 🛠 Стек технологий

* **Язык:** Python
* **Библиотеки:** Pandas, NumPy, Seaborn, Matplotlib, Pathlib, re.
* **Аналитика:** Парсинг директорий (os.walk / rglob), временной анализ (Time Series), объединение данных (merge), расчет Unit-экономики (AOV, Revenue), корреляционный анализ, выявление аномалий, визуализация данных (EDA).

## 🎯 Решенные задачи

1. **Интеграция разрозненных данных:** Написание скрипта для обхода дерева папок и сборки единых датафреймов из сотен локальных файлов orders.csv, order_status.csv и products.csv.
2. **Мультивалютная аналитика:** Обогащение базы данных ежедневными курсами доллара (usd_rate.txt) для пересчета стоимости заказов и расчета реальной выручки в рублях.
3. **Выявление аномалий:** Поиск дней с пиковыми значениями заказов и отмен, дата-расследование причин резкого скачка активности.
4. **Оценка ассортимента:** Определение топ-брендов по выручке и поиск "залежавшегося" товара (неликвида), который захламляет прайс-листы.
5. **Оценка KPI отдела продаж:** Расчет доли каждого менеджера в абсолютной выручке и общем количестве оформленных заказов.
6. **Пет-проект (макроэкономика):** Построение матрицы корреляций между курсом доллара и активностью B2B-клиентов, оценка упущенной выгоды из-за "валютных" отмен.

## 💡 Ключевые бизнес-инсайты

* **Ложный рекорд продаж и сбой CRM:** Аномальный всплеск успешных заказов 14 марта оказался следствием технического сбоя 13 марта. Около 42% "новых" заказов — это полные дубликаты отмененных вчерашних корзин. Отдел продаж не сделал прорыв, а лишь восстановил потерянное из-за бага системы.
* **Лидеры рынка:** Основную выручку компании генерируют 5 брендов: JBL, Heco, Klipsch, Yamaha и KEF. Именно они формируют ядро оптовых продаж.
* **Проблема неликвида:** Выявлены бренды, чья продукция зря занимает виртуальную "полку". У брендов Marantz, Dali, Pioneer и KEF более 50% широкого ассортимента ни разу не добавлялось в корзину за месяц. Требуется пересмотр закупочной матрицы по этим позициям.
* **Упущенная выгода и отмены:** Анализ показал, что колебания курса доллара ведут к существенной упущенной рублевой выручке из-за массовой отмены заказов. Возможная рекомендация для бизнеса — внедрить политику временной фиксации курса для B2B-клиентов на 24 часа после создания заявки в CRM.
* **Прозрачность работы менеджеров:** Выявлен значительный разрыв в KPI сотрудников. Визуализация процента принесенной выручки и количества обработанных заказов дала руководству четкую картину для распределения премий и проведения работы с отстающими менеджерами.

## 📂 Структура репозитория

* `Анализ данных об оптовых продажах аудиотехники.ipynb` — Jupyter Notebook с полным циклом обработки данных, расчетами конверсий и визуализациями.
* `data/orders/` — директория с сырыми логами заказов и статусов, разбитая по датам и менеджерам.
* `data/products/` — директория с сырыми данными по товарам, разбитая по категориям.
* `usd_rate.txt` — исторические данные по курсу доллара к рублю.
* `orders_status.csv` — сгенерированный промежуточный сводный датасет со статусами.
