# 🚕 Аналитика лояльности и поведенческих паттернов сервиса такси (Перу)

## 📋 Бизнес-контекст
Сервис заказа такси в Перу стремится повысить качество обслуживания и удержать как водителей, так и клиентов. Для этого необходимо понимать, как формируются оценки, существует ли конфликт интересов между участниками и что провоцирует отмены заказов. 
**Цель данного проекта:** рассчитать реальные метрики поездок по геоданным, проанализировать культуру взаимного оценивания и проверить гипотезу о влиянии дистанции на отток заказов (Cancellations).

## 🛠 Стек технологий
* **Язык:** Python
* **Библиотеки:** Pandas, NumPy (векторизация расчетов), Seaborn, Matplotlib.
* **Аналитика:** Матричный анализ (Crosstab), Geospatial Analysis (формула Гаверсинуса), EDA, работа с выбросами и фильтрация данных.

## 🎯 Решенные задачи
1. **Предобработка и очистка:** Обработка пропущенных значений в рейтингах, приведение данных к формату, пригодному для аналитики.
2. **Геопространственный расчет:** Реализация векторизованной функции Гаверсинуса для вычисления точного расстояния между координатами старта и финиша.
3. **Анализ матрицы лояльности:** Построение нормированных кросс-таблиц для оценки «взаимной неприязни» и проверки принципа «око за око».
4. **Визуализация распределений:** Использование Boxplots с ограничением осей (Zoom-эффект) для детального изучения дистанций в обход GPS-выбросов.
5. **Анализ статусов поездок:** Исследование структуры завершенных заказов и отмен со стороны клиента и водителя.
6. **Рефакторинг визуального долга:** Замена неинформативных круговых диаграмм на профессиональные Bar Charts с аннотациями значений.

## 💡 Ключевые бизнес-инсайты

* **Опровержение гипотезы «Мести» в рейтингах:** Анализ матрицы частот через `sns.heatmap` показал феноменальную устойчивость водителей. Даже в тех случаях, когда пассажир ставил минимальный балл (**1.0**), более **64%** водителей всё равно оценивали клиента на **5.0**. Взаимная низкая оценка (1 и 1) — крайне редкое явление. Это свидетельствует о высоком профессионализме водителей или их склонности ставить высший балл «по умолчанию».
* **Дистанция не является триггером отмен:** Сравнение распределений через «ящики с усами» показало, что медианная дистанция для завершенных поездок (`drop off`) и отмененных пассажирами (`rider cancel`) практически идентична — около **5 км**. Гипотеза о том, что длинные поездки отменяют чаще, **не подтвердилась**. Причины отмен, вероятно, связаны с временем подачи машины (ETA), а не с дальностью маршрута.
* **Аномалии GPS-данных:** В ходе расчетов были выявлены критические выбросы (дистанции свыше **1000 км** для внутригородских поездок). Это подтверждает наличие технических сбоев при фиксации координат. Для корректного анализа был применен метод ограничения области видимости графиков (`plt.ylim`), что позволило сфокусироваться на реальном поведении пользователей в пределах города.
* **Доминирование позитивного смещения:** Как у водителей, так и у пассажиров оценки «5» составляют подавляющее большинство (свыше 80%). При этом пассажиры являются более строгими критиками: доля низких оценок (1-3) у них стабильно выше, чем у водителей, которые практически всегда ставят высший балл.
