## Аналитик данных — Karpov.Courses

Этот репозиторий содержит мой прогресс обучения и практические задания по программе **Data Analytics от Karpov.Courses**.

В репозитории представлены:

- SQL-запросы и аналитические задачи
- анализ данных на Python
- задачи по теории вероятностей и статистике
- продуктовая аналитика
- практика работы с Airflow
- аналитические проекты, оформленные как портфолио

---

## Структура репозитория

Репозиторий разделён на две основные части:

### learning

Учебные задания и материалы курса.

- `01_sql` — практика SQL и аналитические запросы  
- `02_python` — упражнения на Python и анализ данных (pandas, numpy)  
- `03_git` — основы работы с Git и контроль версий  
- `04_probability` — задачи по теории вероятностей  
- `05_statistics` — статистика и A/B тестирование  
- `06_product_analytics` — метрики продукта и аналитические задачи  
- `07_airflow` — практика построения DAG и работы с Airflow  
- `08_final_project` — финальный проект курса  

---

### portfolio

Аналитические проекты, оформленные как портфолио.

#### SQL проекты

- **Анализ популярности брендов среди клиентов Москвы**
  - расчёт количества заказанных товаров по брендам
  - определение доли бренда в общем объёме заказов
  - ранжирование брендов по популярности
